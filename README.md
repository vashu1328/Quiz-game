# Quiz-game
Quiz App built using HTML, CSS, and JavaScript that allows users to answer multiple-choice questions and shows score at the end
A simple Quiz App built using HTML, CSS, and JavaScript.  
This app allows users to answer multiple-choice questions and displays the final score at the end.

## Features

- Multiple choice questions
- Score calculation
- Next question button
- Simple and clean UI
- Responsive design
- Beginner friendly project

## Tech Stack

- HTML
- CSS
- JavaScript

##  How to Run

1. Clone the repository
2. Open index.html in browser
3. Start the quiz
4. See final score
